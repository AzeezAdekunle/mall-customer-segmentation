# Mall Customer Segmentation using K-Means

## Project Overview
This project applies K-Means clustering to segment mall customers into distinct groups based on their annual income and spending score. Customer segmentation helps businesses understand purchasing behavior and design targeted marketing strategies.

## Dataset
The dataset contains information about mall customers including:

- CustomerID
- Gender
- Age
- Annual Income
- Spending Score

## Methodology

1. Data exploration and preprocessing
2. Feature selection
3. Feature scaling
4. Determining the optimal number of clusters using the Elbow Method
5. Applying K-Means clustering
6. Visualizing and interpreting customer segments

## Results

The model identified several distinct customer groups based on income and spending behavior, including:

- High income, high spending customers
- High income, low spending customers
- Low income, moderate spending customers

These insights can help businesses design targeted marketing strategies for different customer segments.
